# JVK Fotografie Portfolio

Deze website is gebouwd met [Hugo](https://gohugo.io/) en gebruikt het thema [hugo-theme-gallery](https://themes.gohugo.io/themes/hugo-theme-gallery/).

Broncontent en foto's zijn gemigreerd vanaf [jvkfotografie.nl](https://jvkfotografie.nl).

## Lokale ontwikkeling

1. Installeer Hugo Extended.
2. Start de site lokaal:

```bash
hugo server -D
```

## Deploy

Deploy naar GitHub Pages gebeurt automatisch via de workflow in `.github/workflows/hugo.yml` bij push naar `main`.
